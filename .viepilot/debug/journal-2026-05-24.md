### 10. `29b0a35` — feat(aura): real-time bilingual translation mode like Google Translate

**Vấn đề từ sếp feedback:** "Phiên dịch không hợp lý — phải dịch theo thời gian thực như Google Dịch"

**Kiến trúc giải pháp:**
- Gemini audio có latency 2-10s → KHÔNG PHÙ HỢP cho real-time
- Android `SpeechRecognizer` → partial results mỗi ~100ms → câu hoàn chỉnh mỗi 1-3s
- Mỗi câu hoàn chỉnh → 1 coroutine riêng → `AuraService.translateText()` (Gemini text translation, 0.1 temp, 512 tokens)
- Kết quả hiển thị ngay trong `SegmentCard` song ngữ

**Files mới:**
- `RealtimeTranslationManager.kt`: wraps `SpeechRecognizer`, continuous STT, auto-restart sau mỗi utterance
- Tất cả SpeechRecognizer calls wrap trong `Handler(Looper.getMainLooper())` — API requirement

**Compose bugs phát hiện và fix:**
1. `rememberLauncherForActivityResult` bên trong `when` block → vi phạm Compose rules → chuyển lên top-level
2. `RecordMode` enum thiếu `CHAT` state → thêm vào như default value
3. `EaseInOut` không available tất cả Compose versions → đổi thành `FastOutSlowInEasing`

**Pattern thread-safe cho concurrent translation:**
```kotlin
val idx = _liveSegments.value.size              // capture index trước
_liveSegments.value += TranslationSegment(sentence)
launch {
    val translation = translateText(sentence)
    val list = _liveSegments.value.toMutableList()
    if (idx < list.size) list[idx] = list[idx].copy(translation = translation)
    _liveSegments.value = list
}
```

## 🐛 Bug Log — Debug History

### BUG-001: JVM broken — không build được từ terminal
**Triệu chứng:** `Error: could not open E:\Programes\App-Studio\jbr\lib\jvm.cfg`  
**Root cause:** JBR trong `E:\Programes\App-Studio\jbr` thiếu file, chỉ có `bin/` + `lib/modules`  
**Fix:** Dùng JBR đầy đủ tại `C:\Program Files\Android\Android Studio\jbr`  
**Build command:** `JAVA_HOME="C:/Program Files/Android/Android Studio/jbr" ./gradlew assembleRelease`

### BUG-002: ProGuard obfuscate Kotlin Serialization → Gemini parse crash
**Triệu chứng:** Release APK crash khi gọi AI; debug APK hoạt động bình thường  
**Root cause:** R8 đổi tên data class fields → `@SerialName` không khớp → JSON parse fail  
**Fix:** `proguard-rules.pro` thêm `-keep class com.systemleveling.core.ai.**`

### BUG-003: Room schema mismatch MIGRATION_14_15
**Triệu chứng:** App crash khi upgrade từ DB version 14 lên 15  
**Root cause:** `@ForeignKey` trên `CourseEntity` tạo implicit index `parentId`, nhưng `MIGRATION_14_15` không khai báo index này → Room schema hash không khớp  
**Fix:** Thêm explicit `@Index(value = ["parentId"])` vào `@Entity` annotation

### BUG-004: `LocalDate.ofInstant()` crash trên Android < API 34
**Triệu chứng:** App crash trên Android 10-12; hoạt động bình thường trên Android 14  
**Root cause:** `LocalDate.ofInstant()` là API 34-only  
**Fix:** `LocalDateTime.ofEpochSecond(epochMs / 1000, 0, ZoneOffset.UTC).toLocalDate()`

### BUG-005: Compose rules violation — launcher inside `when` block
**Triệu chứng:** Compile warning "Composable calls are not allowed inside the when expression"  
**Root cause:** `rememberLauncherForActivityResult` gọi bên trong `when { else -> }` block  
**Fix:** Cả hai launcher chuyển lên top-level của `NpcChatScreen()`

### BUG-006: SpeechRecognizer — phải chạy trên main thread
**Triệu chứng:** `android.os.NetworkOnMainThreadException` hoặc crash không rõ ràng  
**Root cause:** SpeechRecognizer.create() và startListening() phải gọi từ main thread  
**Fix:** Bọc tất cả calls trong `Handler(Looper.getMainLooper()).post { }`

### BUG-007: Gemini audio API rejects `system_instruction`
**Triệu chứng:** HTTP 400 khi gửi audio request kèm system_instruction  
**Root cause:** Gemini multimodal audio endpoint không chấp nhận `system_instruction` field  
**Fix:** Tạo `GeminiAudioRequest` class riêng không có `systemInstruction`

## 📐 Architecture Decisions

| Quyết định | Lý do |
|-----------|-------|
| SpeechRecognizer thay vì Gemini STT cho real-time | Latency: SpeechRecognizer ~100ms, Gemini audio 2-10s |
| Coroutine per sentence (không batch) | Dịch ngay khi có kết quả, không chờ hết session |
| Index capture trước khi launch coroutine | Thread-safe update: tránh race condition khi nhiều câu dịch đồng thời |
| Gemini `temperature=0.1` cho translateText | Độ chính xác cao nhất, không sáng tạo khi dịch |
| EncryptedSharedPreferences cho API key | AES256-GCM, không thể đọc qua ADB backup |
| `GeminiAudioRequest` tách biệt `GeminiRequest` | API constraint: audio endpoint không nhận system_instruction |
