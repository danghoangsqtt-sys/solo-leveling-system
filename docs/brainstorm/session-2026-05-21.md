## 🏗️ Kiến Trúc Hệ Thống AI Liên Tục

### Luồng vận hành trong 1 ngày

```
⏰ 00:00 / App mở lần đầu
    → 🤖 AI Quest Generator
    → 📋 Sinh 6-10 Daily Quests
    → 📱 Hiển thị Quest Timeline

Trong ngày:
    → ⏰ Nhắc nhở nhiệm vụ (WorkManager)
    → 💧 Nhắc uống nước (mỗi 2h)
    → 📋 User hoàn thành quest
        → 🎁 Reward Engine
            → ✨ +EXP +Gold +SP
            → 🎲 Item Drop (Loot Table)
            → 🏆 Title Progress Check
    → Quest hết hạn không hoàn thành?
        → ⚠️ Penalty Engine → +Debt Points, -EXP

    → 🌙 21:00 Nhắc ngủ sớm
    → 📖 21:30 Nhắc viết nhật ký

⏰ 22:00 / Cuối ngày
    → 🤖 AI Journal Writer → 📝 Tự động viết nhật ký
    → 📊 Daily Summary Generator
    → 📱 Notification: "Xem Tổng Kết"
    → 🖥️ Daily Report Screen
```

### Luồng AI Skill Roadmap

```
🎯 User chọn Goal
    → 🤖 AI phân tích Goal
    → 🌳 Sinh Skill Tree (8-15 nodes)
    → 📋 Sinh Quest liên quan đến skill nodes
    → ✅ Hoàn thành quest
    → 🔮 +SP vào skill tương ứng
    → Đủ SP lên cấp? → 🎉 Skill Level Up! + Unlock nodes tiếp
    → Lặp lại...
```

### 7. Stat Calculation Formula

| Quest Category | Stats Affected |
|---|---|
| fitness | STR, VIT |
| language, study, reading | INT |
| tech, coding | INT, AGI |
| social, communication | CHA |
| meditation, journal | WIS |
| finance | WIS |

Formula: `stat_gain = ceil(quest_exp / 50)`
