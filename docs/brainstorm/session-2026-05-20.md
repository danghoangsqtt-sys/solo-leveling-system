## 1. Project Overview & Meta Info

We have downloaded the full UI design guidelines and screen mockups from Stitch for the project **solo_leveling_system** (ID: `4112373676990532485`).

- **Aesthetic Direction:** Holographic Glassmorphism. Features extreme backdrop blurs (40px-50px), semi-transparent layers, neon-tinted outlines, and a "Hunter's Terminal" / gamified RPG interface look.
- **Typography:** Dual-font structure.
  - **Rajdhani** for game-like HUD headings and caps labels.
  - **Inter** for high-readability body text, logs, and dialogue boxes.
- **Color Palette (Cosmic Void):**
  - **Primary (System Blue):** `#4a9eff` — Standard HUD interactivity and focal status pulses.
  - **Secondary (System Gold):** `#ffd700` — High-tier or epic achievements, status rewards.
  - **Tertiary (System Green):** `#2ed573` — System health, active buffs, and permission alerts.
  - **Background:** `#121222` — Infinite dark indigo void depth.
