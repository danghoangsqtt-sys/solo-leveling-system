## Tổng quan dự án

**Tên:** System Leveling  
**Ý tưởng:** Ứng dụng Android Native biến cuộc sống thực thành game RPG cá nhân, lấy cảm hứng từ hệ thống (The System) trong manhwa Solo Leveling. Người dùng trở thành nhân vật chính, được AI dẫn dắt qua các nhiệm vụ hàng ngày để phát triển: thể chất, trí tuệ, kỹ năng nghề nghiệp, tài chính.

**Phong cách thiết kế:**
- Giao diện tươi sáng, gradient MMORPG (MapleStory, Genshin Impact)
- Đồ họa 2.5D — icon, avatar có chiều sâu
- Particle effects — confetti, glow, sparkle
- Animation mượt 60fps — transitions, micro-interactions
- Dark mode mặc định + Light mode option

## Phân tích 14 Module chức năng

### 1. Onboarding — Khởi tạo nhân vật
- **Bước 0:** Cinematic intro (text glitch, scan line, particle)
- **Bước 1:** Thông tin cơ bản (nickname, avatar, ngày sinh, giới tính, chiều cao, cân nặng)
- **Bước 2:** Khảo sát năng lực — 18-24 câu hỏi cho 6 chỉ số: STR, INT, AGI, VIT, WIS, CHA (thang 1-100)
- **Bước 3:** Chọn Class — 8 lớp (Warrior, Mage, Ranger, Guardian, Bard, Alchemist, Healer, Scholar)
- **Bước 4:** Chọn Goals — mục tiêu dài hạn (có sẵn + custom)

### 2. Home — Status Panel
- Avatar chibi 2.5D + stats bars (STR/INT/AGI/VIT/WIS/CHA)
- EXP bar animated, Level, Gold/Gem, Streak, Debt
- Today's Quests mini card
- Background thay đổi theo level range (Forest→Desert→Ocean→Mountain→Castle→Galaxy)
- Bottom Navigation: 5 tab chính + drawer cho phần còn lại

### 3. Skill Tree — Cây kỹ năng
- Node graph zoomable/pannable (kiểu Path of Exile)
- 7 bậc kỹ năng: Novice → Apprentice → Intermediate → Advanced → Expert → Master → Grand Master
- SP system (Skill Points) tích lũy từ quest completion
- Mỗi Goal → 1 Skill Tree (8-15 skills, 3-5 nhánh)
- AI sinh cây kỹ năng tự động

### 4. Quest System — CORE FEATURE
- AI gen 6-10 quests/ngày cá nhân hóa
- 6 loại quest: Daily, Weekly, Boss, Penalty, Event, Side
- 6 rank: E → D → C → B → A → S (với EXP range tương ứng)
- Timeline dọc kiểu daily planner
- Popup hoàn thành: rewards (EXP, Gold, SP, Item Drop), animation

### 5. Penalty System
- Debt Points: +1 khi fail quest, penalties tiến triển (cảnh báo → mất gold → level down → mất title)
- Penalty Quests: khó 1.5x, không có thưởng, chỉ trả nợ
- UI: viền đỏ nhấp nháy, notification Solo Leveling style

### 6. Inventory — Kho vật phẩm
- 6 tier rarity: Common → Uncommon → Rare → Epic → Legendary → Mythic
- Grid layout 4-5 cột kiểu MMORPG
- Shimmer/glow animation cho Legendary+

### 7. Title System — Danh hiệu
- Title mẫu gắn với achievement (streak, quest count, level, skill mastery...)
- AI sinh title động khi phát hiện pattern hành vi
- Equip 1 title hiển thị dưới tên nhân vật

### 8. Finance — Quản lý tài chính
- Dashboard: Net Cashflow hero card, charts (Bar, Pie, Line)
- Transactions: SmartMoneyInput (NLP parse: "ăn phở 50k" → auto categorize)
- Budget Tracker + Rollover
- Financial Goals + Debt Tracker
- AI Financial Analysis
- Gamification tích hợp (ghi chi tiêu đúng hạn = +EXP, vượt budget = Debt Point)

### 9. Learning Library
- Cây thư mục đệ quy, multi-format viewer (PDF, PPTX, YouTube, GDrive, Note)
- Zen Mode, Note Overlay, Progress Tracking, Tag System
- Liên kết Skill Tree: đọc/xem xong = +SP
- Gamification (đọc 30p = +80 EXP, hoàn thành folder = Boss Quest)

### 10. Journal — Nhật ký
- Rich text editor (Markdown), mood selector, auto-tag theo ngày
- AI tóm tắt tuần/tháng
- UI phong cách sổ tay RPG

### 11. Calendar — Lịch
- Month/Week/Day views
- Quest overlay trên calendar
- Habit Tracker (7 chấm lịch sử)
- Streak counter

### 12. To-Do List
- CRUD + priority (Urgent/High/Medium/Low)
- "Convert to Quest" feature (tùy chọn)

### 13. Settings
- Profile, Goals, Quest Config, Notifications, Theme, AI API Key
- Language (VI/EN), Penalty Config, Finance Config
- Backup & Restore

### 14. AI Integration
- 4 prompt templates chính: Quest Gen, Skill Tree Gen, Title Gen, Finance Analysis
- Fallback offline: quest templates, default skill trees, default titles

## Database Schema

**20 entities** trong Room Database:
- Core: UserEntity, GoalEntity, SkillEntity, QuestEntity
- RPG: InventoryItemEntity, TitleEntity, PenaltyLogEntity, QuestTemplateEntity
- Finance: TransactionEntity, BudgetCategoryEntity, FinancialGoalEntity, DebtEntity
- Library: LibraryNodeEntity, LibraryNoteEntity
- Productivity: JournalEntryEntity, CalendarEventEntity, HabitEntity, TodoEntity

## Decisions

| # | Quyết định | Lý do |
|---|---|---|
| D1 | Platform: Android Native (Kotlin + Compose) | Theo spec, tối ưu performance cho animation/game UI |
| D2 | Architecture: MVVM + Clean Architecture multi-module | Dễ scale, tách biệt feature, testable |
| D3 | Database: Room (offline-first) | Theo spec, không cần backend cho MVP |

| D5 | Navigation: 5 bottom tabs + drawer | 9 screens quá nhiều cho bottom bar |

## Phases

### Phase 1 — Foundation & Core (6 tuần)
- Project setup: Multi-module, Hilt, Room, Navigation, Theme
- Design System: GameCard, colors, typography, animations cơ bản
- Onboarding: Cinematic intro, Survey, Class selection, Goal selection
- Home Screen: Status Panel, stats, EXP bar, level display
- Basic Quest System: AI gen + completion + reward distribution
- Settings: AI key config, basic preferences

### Phase 2 — RPG Systems (6 tuần)
- Skill Tree: Full node graph UI + AI generation + SP tracking
- Inventory: Grid UI + item detail + rarity system
- Title System: Achievement tracking + title gallery + equip
- Penalty System: Debt tracking + penalty quest generation
- Level Up animations + background theme changes
- Notification system: WorkManager quest reminders

### Phase 3 — Productivity Tools (6 tuần)
- Calendar: Month/Week/Day views + Quest overlay + Events
- To-Do List: CRUD + priority + deadline + Convert-to-Quest
- Journal: Rich editor + mood + AI summary
- Habit Tracker: Daily tracking + 7-day history

### Phase 4 — Finance & Library (6 tuần)
- Finance Dashboard: Transaction CRUD + Charts
- Budget Tracker + Rollover, Financial Goals + Debt Tracker
- SmartMoneyInput (AI-powered), AI Financial Analysis
- Learning Library: Folder tree + PDF/PPTX/YouTube/GDrive viewers
- Note overlay + Zen Mode + Progress tracking
- Library ↔ Skill Tree integration

### Phase 5 — Polish & Ship (4 tuần)
- Animation polish (Lottie, particle effects)
- Performance optimization
- Đa ngôn ngữ VI/EN
- Backup/Restore + Google Drive
- Offline quest templates
- UI testing đa thiết bị
- Play Store preparation
